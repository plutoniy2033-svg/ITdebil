# SASpdf

Веб-редактор PDF-файлов с полным набором инструментов для работы с документами.

## Возможности

| Функция | Описание |
|---------|----------|
| **Просмотр** | Открытие и просмотр PDF с масштабированием и навигацией по страницам |
| **Выделение текста** | Выделение и копирование текста из документа |
| **Добавление текста** | Размещение нового текста в любом месте страницы |
| **Правка текста** | Исправление текста непосредственно в документе (наложение) |
| **Заметки** | Добавление текстовых заметок-аннотаций |
| **Подпись** | Рисование и размещение электронной подписи |
| **Изображения** | Добавление, удаление и изменение размера изображений |
| **Замазка (Redaction)** | Скрытие конфиденциальной информации чёрными блоками |
| **Защита** | Установка пароля, запрет печати, копирования и редактирования |

## Быстрый старт

```bash
cd SASpdf
npm install
npm run dev
```

Приложение откроется по адресу `http://localhost:5173`.

## Сборка

```bash
npm run build
npm run preview
```

## Стек

- **React 18** + **TypeScript** — интерфейс
- **Vite** — сборка и dev-сервер
- **pdf.js** — рендеринг и выделение текста
- **pdf-lib** — редактирование, подписи, изображения, шифрование

## Использование

1. Нажмите **Открыть** и выберите PDF-файл
2. Выберите инструмент на панели инструментов
3. Кликните на странице для размещения элемента
4. Перетаскивайте и изменяйте размер объектов в режиме **Выбор**
5. Настройте свойства в боковой панели
6. **Сохранить** — скачать отредактированный PDF
7. **Защита** — установить пароль и ограничения
